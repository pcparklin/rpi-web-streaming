Web gui for streaming video to youtube | facebook on raspberry pi

This project includes pre-compiled ffmpeg and mmal_video_record.
This project is written in python.

Dependencies
------------

flask, flask_wtf, lxml


Using
-----

Run commands below on raspberry pi:
```
sh db.sh
python web.py
```


Reference
---------
